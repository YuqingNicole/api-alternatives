# Google Maps API 变更记录

## 2025

**2025-03** Google Maps Platform 重新定价：
- 移除每月 $200 固定信用额
- 改为按 SKU 的免费额度（如 Essentials SKU 每月 10,000 次免费加载）
- 推出订阅计划：Starter ~$100/月（50K 次调用）、Essentials ~$275/月（100K 次）、Pro 更高
- 小流量用户受影响较小，中大流量用户成本上升

## 2018

Google Maps 首次重大定价调整，将多个 API 合并计费，部分场景成本上涨 14 倍，引发大规模迁移潮（Mapbox 等受益）。
