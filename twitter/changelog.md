# Twitter / X API 变更记录

## 2026

**2026-02** 官方 X API 彻底移除免费层，全面 pay-per-use。Pro 计划 $5,000/月，100 万条推文/月。
**2026-03** GetXAPI 部分 POST 接口临时不可用（官方 API 变更导致），团队正在修复。

## 2023

**2023-02** Twitter 移除免费 API 访问，开始收费。
**2023-Q2** Apollo、Reddit is Fun 等依赖免费 API 的第三方应用关闭，第三方替代方案市场爆发。
